応募状況について

応募状況

応募状況の説明

応募者未確認

応募者が作成した自分の応募情報又は研究計画調書のPDFファイルを確認していない状態です。応募者の確認が完了しないと、所属研究機関担当者は承認・却下を行うことはできません。

所属研究機関受付中

応募者から応募情報又は研究計画調書が提出されました。応募情報又は研究計画調書の内容を確認の上、承認（チェックリスト確定）又は却下を行ってください。

領域代表者確認中

研究計画調書が確認されました。研究計画調書が領域代表者へ送られた状態です。

領域代表者確定済み

領域代表者が領域計画書を確定した状態です。

領域代表者却下

領域代表者が研究計画調書を却下した状態です。

領域代表者受付締切

領域代表者が領域計画書の受付を締め切った状態です。

領域代表者所属研究機関受付中

応募書類が領域代表者所属研究機関に受付られた状態です。

学振受付中

所属研究機関担当者が応募情報又は研究計画調書を承認（チェックリスト確定）し、応募情報又は研究計画調書が日本学術振興会へ送られた状態です。

却下

所属研究機関担当者（又は部局担当者）が応募情報又は研究計画調書を却下した状態です。応募情報又は研究計画調書を却下すると、日本学術振興会に応募情報又は研究計画調書は送信されません。

学振受理

日本学術振興会が応募情報又は研究計画調書を受理しました。
