応募状況

応募状況の説明

仮領域番号確認待ち

応募者が仮領域番号発行情報を登録しました。仮領域番号発行の確認又は却下を行ってください。

仮領域番号発行済み

所属研究機関担当者が仮領域番号発行を確認し、仮領域番号が発行された状態です。

仮領域番号発行却下

所属研究機関担当者が仮領域番号発行を却下し、仮領域番号が発行されなかった状態です。

領域代表者未確認

応募者が作成した自分の領域計画書のPDFファイルを確認していない状態です。応募者の確認が完了しないと、所属研究機関担当者は承認・却下を行うことはできません。

所属研究機関受付中

応募者から応募書類が提出されました。応募書類の内容を確認の上、承認又は却下を行ってください。

学振受付中

所属研究機関担当者が応募書類を承認し、応募書類が日本学術振興会へ送られた状態です。

却下

所属研究機関担当者（又は部局担当者）が応募書類を却下した状態です。応募書類を却下すると、日本学術振興会に応募書類は送信されません。

学振受理

日本学術振興会が応募書類を受理しました。
