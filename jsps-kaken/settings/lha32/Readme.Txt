【ソフト紹介】

　Win95 で半角カタカナを含むディレクトリにインストールすると正常動作しない
  バグに対応しました．
  その他，バグ対応．

【注意】
　吉崎栄泰氏の LHA32.EXE とは別物です(間違えた方はゴメンなさい)．
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
【言い訳】
　既に，吉崎氏が LHA32.EXE を公開してるとは知らずに，同じ名前で登録してしま
いました(名前を考えるのが面倒だった(^_^;))．吉崎氏オリジナルの LHA32.EXE を
希望の方は FLABO あたりを探してください．

【特徴】
　Lha32.Exe は WinNT,Win95 の Console で，LZH 形式書庫の圧縮及び解凍を行う
ソフトです． 

  WinZIP の外部プログラムとして利用できます(^_^)．

  Lha32.Exe は，なにもしません(^_^;)．コマンドラインを UNLHA32.DLL に渡すだ
けです

【動作条件】
・UNLHA32.DLL Ver 0.36 以降がないと動作しません．
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
・UNLHA32.DLL は，別途ダウンロードしてください．
  UNLHA32.DLL は FWINNT DL#11 にあります(97/10/26 現在)


【インストール】===========================================================

 LHA32106.EXE を実行すると解凍をはじめます．
 適当なディレクトリにインストールしてください．

【使用方法】===============================================================

 WinNT，Win95 のコンソール画面から以下の様式で使用してください．
 >Lha32 [<command>] [[-<switch>[+|-|0|1|2]]...] <archive_file_name>[.LZH]
          [[<base_directory_name>\] [<path_name>|<wild_name>]...]]...

 Lha32.Exe のコマンドラインの説明は，UNLHA32.DLL 付属の COMMAND.TXT を参照
 してください．

 UNLHA32.DLL の使用方法は，UNLHA32.DLL 付属の UNLHA32.TXT を参照してください

【注意事項】===============================================================

 MS-DOS, Win3.1, Win32S, OS/2 では動作しません．

 UNLHA32.DLL Ver 0.62a まで動作確認しております．
