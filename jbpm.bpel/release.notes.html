Release Notes - JBoss jBPM BPEL - Version 1.1 beta 2

The enhancements and related JIRA issues appear next.

Bug

  • [BPEL-92] - While activity implementation not to use runtime stack for activity trace
  • [BPEL-191] - automatic node creation does not preserve source prefix
  • [BPEL-192] - hibernate mapping for sybase.
  • [BPEL-194] - Invoking two different web services in a row causes nullpointer exception.
  • [BPEL-195] - namespace propagation does not produce desired results
  • [BPEL-196] - jbpm and bpel services do not play together well
  • [BPEL-197] - integers are represented with a decimal point when used as source content
  • [BPEL-199] - BPEL reader does not contain logic to parse 1.1 fault handlers.
  • [BPEL-206] - activity readers file is reparsed every time a bpelReader is instantiated
  • [BPEL-207] - module definitions file is reparsed every time a bpelDefinition is instantiated
  • [BPEL-210] - compatibility broken with the axis-based stack in jboss 4.0.3 and below

Feature Request

  • [BPEL-181] - detect absence of required partner link role
